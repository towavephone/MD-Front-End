## Pull requests  

请按照如下步骤构建项目：

1. git pull项目.
2. 在项目根目录运行npm install或cnpm install
3. 开发环境运行npm run dev，访问[localhost](http://localhost/)
4. 生产环境运行npm run prod，复制dist、build、index.html到服务器目录下
